# 测试 React 组件我们采用 Enzyme 工具库，它提供 3 种组件渲染方式：

  1. Shallow ：不会渲染子组件
  2. Mount : 渲染子组件，同时包含生命周期函数如 componentDidMount
  3. Render : 渲染子组件，但不会包含生命周期，同时可用的 API 也会减少比如 setState()

# snapshot
